# 🏥 PrevImagem — Site Institucional

Site institucional da Clínica PrevImagem, desenvolvido como projeto prático da **Residência Tecnológica Porto Digital**, programa **RiseUp UNIT** (Rocketseat + Universidade Tiradentes), 2026.

---

## 📋 Sobre o Projeto

A PrevImagem é uma clínica especializada em diagnóstico por imagem e estética localizada em Paulista, PE. Este projeto entrega o site institucional da clínica com foco em clareza de informações, experiência do usuário e design responsivo.

### O que o site apresenta:
- Seção **Exames e Serviços** (Ultrassom, Raio-X, Mamografia)
- Seção **Estética** (Preenchimento Labial, Métodos de Emagrecimento, Gerenciamento de Pele)
- Seção **Médicos Especialistas**
- Seção **Quem Somos**
- Seção **Convênios**
- **Navbar** com links de navegação e redes sociais
- **Footer** completo
- **Chatbot** de atendimento via WhatsApp

---

## 🚀 Tecnologias Utilizadas

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [Vite](https://vitejs.dev/) — ferramenta de build e desenvolvimento
- [Tailwind CSS](https://tailwindcss.com/) — framework de estilização
- [JavaScript ES6+](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)

## 🧩 Componentes

| Componente | Descrição |
|---|---|
| `Navbar.jsx` | Barra de navegação com logo, menu e redes sociais |
| `QuemSomos.jsx` | Seção institucional com missão, visão e valores |
| `ExamesSection.jsx` | Seção de exames e serviços com cards |
| `EsteticaSection.jsx` | Seção de procedimentos estéticos com cards |
| `MedicosSection.jsx` | Seção de especialistas e médicos com cards |
| `CardSections.jsx` | Card reutilizável usado nas três seções acima |
| `BotaoSection.jsx` | Botão reutilizável "Ver todos os..." |
| `Convenios.jsx` | Grid com logos dos convênios aceitos |
| `Chatbot.jsx` | Chatbot de atendimento via WhatsApp |
| `FooterComponent.jsx` | Rodapé com informações e links |

## ⚙️ Como Rodar o Projeto

**Pré-requisitos:** Node.js instalado

```bash
# Clone o repositório
git clone https://github.com/aghata-guerra/PrevImagem.git

# Acesse a pasta
cd PrevImagem

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```
